## 💡프로젝트 주제 
"song" 포털 사이트의 카페 서비스

## 프로젝트 소개
- "song" 이라는 가상의 포털 사이트가 제공하는 카페 서비스
- 카페에 관한 여러기능들을 제공

## 주요기능
### 카페홈 화면
- 카페 검색
- 카페 생성
- 상단 메뉴바

### 카페메인 화면
- 게시글 검색
- 카페 정보
- 사용자 정보(작성한 글, 댓글 작성한 글)
- 로그인 상태 X -> 로그인 버튼
- 로긍인 상태 O -> 카페 가입버튼(카페 가입 X) or 게시글 쓰기 버튼(카페 가입 O)
- 카페 메인 관리자 모드(게시판 생성, 게시글 삭제, 사용자 강퇴)

### 게시글 화면
- 사용자와 작성자가 같으면 수정, 삭제 버튼 보임
- 댓글 쓰기, 수정, 삭제

### 회원가입
- 아이디, ~~~ 규칙에 맞아야지 성공
- 회원 가입 성공후 song 포털 사이트 화면 이동
  
### 로그인
- 로그인 성공후 로그인 요청한 곳으로 이동
